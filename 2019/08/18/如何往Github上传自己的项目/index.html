如何往Github上传自己的项目

发表于 2019-08-18 | 本文热度 ℃
  • Step 1.进入本地目录
    利用GitBash cd进入当前项目文件地址，可以通过pwd检测
  • Step 2.建立git仓库
    输入git init，成功的标志就是该目录下会多一个.git的隐藏文件夹
  • Step 3.将文件添加到仓库
    git add 指定文件名 或git add –all 将所有的文件全部添加
  • Step 4.将文件提交注释
    git commit -m “修改说明”，可以通过git status查看状态检查是否成功
  • Step 5.将本地仓库关联到github并上传
